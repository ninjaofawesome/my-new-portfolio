Old Fashioned

A homemade boilerplate for Bourbon, Neat and Bitters on Compass.

More

Here's What Happened.

The Setup

Old Fashioned was a project that I put together as I wanted to learn how to use Bourbon, Neat and Bitters and I thought it would be nice to have a project built with Compass, which I also very much enjoy using.

Since it was a fun project for my own learning, I thought it might be nice to create a boilerplate so that others might be able to get up and running for rapid prototyping or other front end work without Bootstrap or Foundation.

The Plan Of Attack

I decided to set this project up for anyone who had never really used Bourbon, Neat or Bitters before by setting up a front end scaffold where the user could drop in components as they needed them into the partials called styles under bourbon and mymixins under the Bourbon directory. This is only a rough guide- additional partials can be added as necessary.

Then I created an index page to ramp the user up even faster with these components. This page is really just to show how all the working parts fit together roughly.

The Solution

The end result was a concise, partially responsive boilerplate perfect for low-complexity projects. I put the work onto a public repository and created a github page to demo the project as well as to share.
