Here's What Happened.

The Setup

Cinna Liselott, a dear friend of mine, is a New York City based puppeteer. She is fortunate to have a full time position creating and performing puppet shows throughout the city, but wanted to have a site of her own as she wanted to expand her horizons through children’s puppet parties.

I was interested in working with Cinna as I wanted to gain some more experience with parallax scrolling, and I felt that she had a unique product that deserved a unique site.

The Plan Of Attack

For this project, Cinna only needed a way for parents to see what services she offered and a way to contact her, that’s about it. Since I was more interested in creating a parallax effect, I decided to keep it simple and focus more on the functions and to keep it all client side.

I decided to use Bootstrap for a CSS framework out of preference, and use SASS partials using a widget-based CSS over object oriented CSS. All of the movement I decided to handle with hand written jQuery (rather than using something pre-written) so that I could figure out how parallax scrolling really worked.

The Solution

The end result was a simple, single page site with aspects of parallax scrolling. (The effect looked similar, but the background doesn’t actually scroll slower than the foreground.) For the more information boxes, I used some jQuery to show and hide elements.

Also, as there was no actual budget for this project and it was a single page application, I was able to deploy it as a github page with its own domain name.
